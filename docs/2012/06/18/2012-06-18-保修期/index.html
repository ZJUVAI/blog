2008年访问 UNCC可视化中心花絮

年前，通过小丁的介绍向非著名可视化专家老鲁卖了一名男丁小俞。

本次过来，还在一年保修期以内，进行了力所能及的保修。

此男的新家，电视只能收一个台，空调的作用范围只有两米，与高速公路的垂直距离只有10米，因此礼节性地拜访吃饭后，落荒而逃。
